Welcome to the HTML and CSS Documentation website!

This website is a comprehensive documentation of everything I have learned from the Interneting Is Hard tutorial, and any other source I have found elsewhere (particularly the MDN Web Docs, a very helpful resource for web developers). From every div, header, and flexbox feature, I will document what and how to make different features work for future reference. I will try to implement as many of these features into this website itself in order to practice, experience, and learn!
