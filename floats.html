Floats

Floats allow us to deviate from the purely vertical flow of the screen. Using floats, you can put block-level elements side-by-side, thus opening the door for sidebars, multi-column pages, grids, and magazine-style articles with text flowing around an image.

Floating an Element

The CSS float property gives you control over the horizontal position of an element. "Floating" a sidebar to the left, for instance, aligns it to the left side of the parent container. It also tells surrounding elements that they can flow around the sidebar instead of beginning underneath it.

float: left;

You can also align the float element center or right, or revert back to default flow with none.

Fixing Float Incorporation: Clearing Floats

When some elements are floated, the browser doesn't count the elements to the parent element's height, and anything below it that is not floated will move up. To "clear" this feature, add clear: both; to any element that ignores floated elements. This forces it back into the default vertical flow of the page and adds the floated elements into the parent element. The both signifies that you are clearing both left and right floats. You can also specify between left and right if you so choose.

Fixing Float Incorporation: Hiding Overflow

If the element right below the floated elements is not within the parent element, clear: both will not work. To add the floated elements back into the parent element, add overflow: hidden; to force the container to recognize the height of any floated elements it contains.

This feature can also be used to prevent text from wrapping an image. In other words, it breaks the magazine-style layout if you need it to.

Multi-column Layout

Floats can be used to create columns of equal width. Style each <div> so that it contains float: left and set the width to the corresponding percentage, depending on how many columns you want. Take any margin space into consideration, as well.

For example, to make 3 columns, style them like this:

.column {
   float: left;
   width: 31%;
   margin: 20px 1.15%;
   height: 160px;
   background-color: #B2D6FF;
 }

The percentage values allow for the columns to accommodate to the width of the parent element or the browser.

Floats for Grids

If you add more elements to a container and it runs out of room to stack them horizontally, they will go down to the next line. Remember to use the overflow: hidden feature if the added elements are no falling within the parent container.

Floats and Images

To create a magazine-style layout with an image and text, just float the image to the left. Don't forget to add padding and margins to space out the content.
