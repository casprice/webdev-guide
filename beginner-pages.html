Beginner Pages

<head> and <body>

The <head> feature of a web page's HTML contains abstract content of the webpage, such as the page title, the CSS links/specific stylesheet, and the type of character encoding implemented (in most cases, UTF-8 is used).

Meanwhile, the <body> feature contains the actual contents that the user sees on the screen, from headings, to paragraphs, to images, and more.

<title>

This contains the title of each webpage, which is displayed in the tab of your page and on Google search engine results.

<p>

This marks each distinct paragraph on the web page.

Headings

Headings titles displayed on the page. They are marked by six levels, from <h1> to <h6>. These elements have tree-like structures, with higher-value headings branching off from lower-value headings. This doesn't really make sense but you get the point.

<ul>, <ol>, and <li>

Unordered Lists

  • Unordered lists are surrounded by the tag <ul>.
  • They list items without order, and the bullets can be replaced with other characters in the CSS.
  • These tend to be used for navigation and side panels.

Ordered Lists

  1. Likewise, ordered lists are surrounded by the tag <ol>.
  2. These are numbered, or can be organized in another form of ordering by altering the CSS.
  3. Ordered lists are used for lists that need step-by-step procedures, tables of contents, and numbered items.
  4. The tag <li> signifies a list item. These are the only elements accepted by unordered and ordered lists.

<em>

The inline tags <em></em> wrap around a word or phrase in a paragraph that you want to draw attention to. By default, it is displayed as italicized text, but can be altered in the CSS.

<strong>

The inline tags <strong></strong> wrap around a word or phrase in a paragraph that you want to draw attention to. By default, it is displayed as bold text, but can be altered in the CSS.

<br> or <br/>

This tag signifies a line break. It can be used within a paragraph element to put the following content on a new line.
This element should not be abused. If you want to add space after a paragraph, increase the padding or the margin.

<hr/>

This tag signifies a horizontal rule, or a thematic break between elements. It has a default appearance as a horizontal line, but can be altered in the CSS to not show in order to create more space between sections.
