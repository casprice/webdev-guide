Links and Images

<a>

This tag stands for "anchor" and creates links. On its own, the <a> doesn't do much. To add actual meaning to the content it contains, you need attributes into the opening tag.
For example, the href attribute tells the browser that it is a link, and renders the text blue and underlined.

Types of Links

Absolute Links

Absolute links refer to a separate web resource. They typically contain a "scheme" (ex. https://), a "domain" (ex. developer.mozilla.org), and a "path" (ex. /en-us/docs/web/html).

Relative Links

Relative links refer to a file in your website, such as another webpage. It's implied that the scheme and domain name are the same as the current page, so you only supply the path.
To refer to a parent folder, use the .. syntax.

Root-Relative Links

Root-relative links are relative to the "root" page of your website (such as the website domain itself). This is represented by simply referring to '/' in the href attribute.

Link Targets

The target attribute of a link defines where to display the page when the user clicks the link. For example, citing '_blank' will open the link in a new tab or window. This is not incredibly secure, as the website that the link directs you to will be able to see the website you came from.

Images

Images are included in web pages with the <img/> tag and its src attribute, which points to the image file you want to display.

JPG Images

JPGs are used for handling large color palettes without exorbitantly increasing file size. This is great for photos and images with lots of gradients in them.

GIF Images

GIFs are used for simple animations, but have a limited color palette. They do not allow for semi-opaque pixels.

PNG Images

PNGs are used for icons, technical diagrams, logos, etc. They are not great for photos or anything animated.

SVG Images

SVGs are vector-based rather than pixel-based and can scale up or down to any dimension without loss of quality. This makes them great for responsive design.

Image Dimensions

The width and height attributes set an explicit dimension for an image. You only need to use one of them, as one will properly scale the image proportionally. Image dimensions should typically be set with CSS rather than HTML.

alt

The alt attribute in image tags defines a "text alternative" to the image being displayed. It is used for search engines and users with text-only browsers (like text-to-speech software).

Document Language

Every webpage should have <html lang='en'> in the HTML tag.

Character Sets

Every webpage should have <meta charset='utf-8'/> in the <head> tag. This defines the character set, thus allowing special characters to render correctly.
