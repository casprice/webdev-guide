Flexbox

Flexbox is a type of CSS used for organizing multiple items within a container. Thus, it uses two types of boxes: "flex containers" and "flex items". Flex containers group a bunch of flex items together and define how they are positioned.

Flex Containers

To turn an HTML element into a flex container, style its CSS to say display: flex;. This enables the flexbox layout, but nothing else until you tell the container how to display its items. Another option for this would be display: inline-flex;.

Horizontal Alignment of Flex Items

In order to define the horizontal alignment or distribution of a flex box's items, use the justify-content property.

justify-content: center;

Options for justifying the content are:

  • center - centers the contents within a flexbox.
  • flex-start - aligns the contents of a flexbox to the left.
  • flex-end - aligns the contents of a flexbox to the right.
  • space-around - equally distributes the amount of space around each item in a flexbox.
  • space-between - equally distributes the amount of space between each item in a flexbox.

Vertical Alignment of Flex Items

In order to define the vertical alignment of a flex box's items, use the align-items property.

align-items: center;

Options for vertical alignment:

  • center
  • flex-start - aligns flex items at the top edge of the flex container.
  • flex-end - aligns flex items at the bottom edge of the flex container.
  • stretch - stretches the flexbox items across the entire height of its container.
  • baseline - aligns flex items along their content's baseline.

Wrapping Flex Items

To render items as a grid, use the flex-wrap property. Its options are either wrap; or nowrap;.

flex-direction

This property specifies whether the flex items are drawn vertically, in a column, or horizontally, in a row.

To switch around the order of the flexbox items, use flex-direction: row-reverse; or flex-direction: column-reverse;.

order

Adding an order property to a flex item defines its order in the container without affecting surrounding items. Its default value is 0, and increasing or decreasing it from there moves the item to the left or right, respectively. This feature works across row/column boundaries, even if they appear on different rows. Note that only positioned elements (see Advanced Positioning) pay attention to their order, also known as thier z-index property.

Flex Item Alignment

If you want only certain contents of a flexbox to be aligned differently than the rest, you can use align-self on the individual element(s) to override the align-items value from its container.

align-self: flex-end;

This property is given the same alignment values as the align-items property.

flex

The flex property defines the width of individual items in a flex container. It gives flex items flexible widths. Assigning each flex item with the default flex: 1 gives all of the flex items equal flex. However, if you assign one flex item with flex: 2, that item will grow twice as fast as the others.

To create static item widths, flex: initial allows the flex item to have an explicit width property.

Flex Items and Auto-margins

To avoid creating so many <div> containers to push some elements to the left and others to the right, using auto-margins eats up all of the extra space in a flex container, so that items are no longer distributed equally.

margin-left: auto;

This moves the element, as well as any following items, to the right side of the container.
