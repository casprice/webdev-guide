Learning CSS

CSS Rules

CSS rules begin with a selector that defines which HTML elements it applies to. For example, if you wanted to edit the body of the web page, you would type:

body {
   color: #FF0000;
 }

color

This is a property that determines the text color of whatever HTML elements have been selected. It accepts hexadecimal and rgb values.

Linking a CSS Stylesheet

Include <link rel='stylesheet' href='styles.css'/> in the head of your HTML file to link the CSS stylesheet to it. Rename the href reference to name of the CSS file. The rel attribute defines the relationship between the resource and the HTML document.

background-color

Pretty self-explanatory. Usually pick text and background colors that do not have high contrast (so not pure black vs. pure white).

font-size

Change the font size of different elements on your web page. This can be done in px or units of em. The em unit scales to match the base font however many times the size.

font-family

This property defines multiple values for the typeface in double quotation marks, and the browser will try to use the fonts listed in the precedence you have ordered them.

list-style-type

For unordered and ordered lists, specifying the list-style-type in the CSS allows you to change bullet or numbered icons, or remove them completely.

text-decoration

This property determines whether text is underlined or not. This is typically used on links to remove the default underlining by setting text-decoration to none. You can also use line-through on this property to create a strikethrough feature on your text.

text-align

This property defines the text alignment, which can be left, right, center, or justify.

font-weight and font-style

The font-weight defines the "boldness" of the text, while font-style indicates whether it's italicized or not.

The CSS Hierarchy

The different forms of styles that can be used on a webpage are organized like so from highest to lowest precedence:

  1. Inline styles
  2. Page-specific styles
  3. External stylesheets
  4. User-defined stylesheets
  5. The browser's default stylesheet

In the same effect, a page-specific style would qualify using the <style></style> tag in the <head> of the HTML file and creating styles that will only be used on that one web page.

If you link multiple stylesheets to a web page, the stylesheet(s) lower down in the list will override any stylesheets above it.
