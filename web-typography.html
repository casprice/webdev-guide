Web Typography

Web Typography deals with basic CSS text properties like what font to use and whether it should be italic or not, as well as features such as the space between and around letters, words, and lines.

Web Fonts

To get free web fonts online, go to Font Squirrel or Google Fonts. You can get premium fonts at Fontspring.

Regarding file type, .otf and .ttf files are both web and desktop fonts. Desktop fonts contain extra functionality useful for graphics editing programs like Adobe Illustrator. Meanwhile, .woff files are designed specifically for the needs of the modern web. Be sure to download or purchase the web font version of the fonts you want to use—not just the desktop version.

Locally Hosted Web Fonts

There is a three-step process to adding a locally hosted web font to a project:

  1. Download a web font and add it to your project.
  2. Embed the web font in your stylesheet.
  3. Use the font elsewhere in your stylesheet.

To download a font from Font Squirrel, be sure to click the Webfont Kit tab, not the Download TTF button, and download the WOFF version. Unzip the file after downloading, and copy all of the font face files you want into the project. Remember, if you don't add the genuine versions of bold or italics, the browser will synthesize it as a lower-quality version when you use <strong> and <em>.

To embed the WOFF file in the web page, put the following at the top of the stylesheet.

@font-face {
  font-family: 'Font Name';
  src: url('filename.woff') format('woff');
}

To add multiple font faces into the style, make multiple of these at-rules, specifying the font-style and font-weight for each, giving the italicized version a font-style of italic and bolded versions a font-weight greater than 300. This way, you are essentially describing each font that you add.

Now you can use 'Font Name' as the font-family anywhere on your stylesheet. Specify the font-weight when you add the regular version of a font to the body, and likewise stylize em with font-style: italic; and strong with font-weight: bold;. Note that the default values for normal is 400 and for bold is 700 for font-weight. This style can take any other boldness level, as well.

Externally Hosted Web Fonts

This is an easier method to adding web fonts to your website. Using Google Fonts, select the fonts you want to add. In the pop-up menu, copy the <link> provided and paste it into the <head> of the webpage you wish to use the font on, above the <link> element that directs to your stylesheet. If you want to add a font later on, put the name of the font at the end of the link, dividing the font names with | and using + instead of spaces.

Now, you can specify the font-family for any text on the webpage.

Paragraph Indents

The most important thing to remember about formatting paragraph indents for blocks of text is to either use a first-line indent or a margin between the paragraphs--never both.

Paragraph indents are styled with the text-indent rule. Usually, the first paragraph after a heading is given a 0 indentation, while subsequent paragraphs have indentation values such as 1em.

Of course, you can add a separating margin with the margin-bottom rule.

Text Alignment

Most of your text is set as the default text-align: left;, especially if the lines run long. Of course, you can always set this to center or right for shorter lines of text (or right for languages that read right-to-left). You can also justify the text, but try to avoid creating blocks of text with awkward spacing.

Line Spacing

The line-height property alters the line spacing of lines in the same paragraph or list.

Line Length

As you already know, you can change the line length of text by altering the width or the margin and padding of the element. A good rule-of-thumb is to limit the number of characters on a single line to around 80.
